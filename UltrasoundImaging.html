A-Mode and B-Mode Beam-forming for Ultrasound Imaging

University of Waterloo · 2021

This project focused on processing and visualizing ultrasound imaging data using both A-mode (amplitude mode) and B-mode (brightness mode) techniques. Implemented entirely in C++, the program handled raw ultrasound data through a custom-designed linked list structure to efficiently store and manage scan lines. Data points within each scan were sorted using a bubble sort algorithm to enhance signal clarity and identify peak echoes.

In A-mode, the amplitude of echo signals was plotted over time to represent object depth and location. In B-mode, multiple A-mode signals were combined and visualized in 2D, producing grayscale images that reveal spatial positioning and density variations of internal structures. The final B-mode display was overlaid with measurement markers and spatial grids to allow accurate distance estimation.

This project demonstrated real-time signal processing, memory management, and algorithm implementation, offering insights into medical ultrasound system development using low-level programming and data structuring.

Key Results

A-mode Signal Peaks:

  • Captured echo amplitude over time to determine depth of reflectors
  • Peaks in the plot represent interfaces between different materials
  • Used bubble sort to isolate dominant echo responses

B-mode Visualization:

  • Combined A-mode scan lines into a 2D grayscale image
  • Visualized internal structures based on reflection strength
  • Grid overlay allows accurate spatial measurement of features

Linked List Data Structure:

  • Implemented in C++ to manage dynamic A-mode scan line storage
  • Each node stored echo samples and metadata for a single scan line
  • Enabled efficient memory handling and sequential reconstruction of B-mode data
