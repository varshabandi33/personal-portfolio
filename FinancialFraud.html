ScamSafe — Financial Fraud Detector

University of Waterloo · 2024

ScamSafe is an AI-powered assistant that protects older adults from financial scams by monitoring texts, emails, and card activity for suspicious patterns. Built with modern cloud tooling and trained on public SMS/email and credit-card fraud datasets, it flags anomalies in real time, learns from user feedback, and can pause risky transactions for review. A simple UI and privacy-first design make proactive protection accessible to non-technical users.

Real-Time Scam & Message Screening

  • Supervised classifier scans SMS/email content for phishing and fraud cues
  • Immediate, plain-language alerts with recommended next steps
  • Continuously improves with user feedback to reduce false positives

Adaptive Anomaly Detection for Transactions

  • Clustering/anomaly models flag unusual card activity in real time
  • Optional “pause-to-confirm” flow for high-risk transactions
  • Modular pipeline for data prep → model train/test → deployment

Privacy & Transparency by Design

  • Minimal data collection with explicit consent and opt-out
  • Encryption-in-transit/at-rest for messages and transaction metadata
  • Clear, accessible privacy explanations to build user trust
